== Description ==

Easy way to disables all wordpress, themes, plugins, core updates & notifications in wordpress.


== Installation ==

1. Download the plugin and unzip it.
2. Upload the folder disable-all-wordpress-updates-notifications/ to your /wp-content/plugins/ folder.
3. Activate the plugin from your wordPress admin.
4. Done.
